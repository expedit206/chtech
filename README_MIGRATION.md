# Sasayee - Vue 3 + Tailwind CSS

Plateforme de marché technologique transformée en Vue.js 3 avec Tailwind CSS.

## Installation et démarrage

```bash
# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev

# Construire pour la production
npm run build
```

## Structure du projet

```
src/
├── components/          # Composants réutilisables
│   ├── Header.vue
│   ├── CategoryCard.vue
│   ├── ProductCard.vue
│   └── BlogCard.vue
├── views/              # Pages principales
│   ├── Home.vue
│   ├── Blogs.vue
│   └── DetailProduit.vue
├── router/             # Configuration du routeur
│   └── index.js
├── App.vue             # Composant racine
├── main.js             # Point d'entrée
└── style.css           # Styles globaux (Tailwind)
```

## Fonctionnalités

- 🎨 Design moderne avec Tailwind CSS
- 🚀 Performance optimisée avec Vue 3 et Vite
- 📱 Responsive design
- 🎯 Routage côté client avec Vue Router
